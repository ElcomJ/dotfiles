Linux and Neovim configurations:


1 - JAVA_HOME and PATH
2 - Create .fonts folder in your home[~], download Hack Nerd Font in Nerd Fonts, and extract in .fonts
3 - Use Hack Nerd Font in terminal
4 - Install xclip (sudo apt install xclip) for use neovim and system clipboard
5 - Install ranger (sudo apt-get install ranger) and trash-cli (sudo apt install trash-cli)
6 - Install Rust
7 - Install ZSH + Oh My ZSH + Powerlevel10k
8 - set "typeset -g POWERLEVEL9K_SHORTEN_STRATEGY=truncate_to_last" in .p10k.zsh
9 - enable rust-version in .p10k.zsh


Git Config:

  - git config --global user.name "Your user name"
  - git config --global user.email email_here
  - git config --global core.editor neovim


Diffview Log:

  - cd .cache/nvim | touch diffview.log
